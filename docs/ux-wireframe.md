# 전체 페이지 와이어프레임

## 페이지 구조 개요
단일 페이지 애플리케이션(SPA) - 헤더/상품 그리드/푸터 3단 구조

---

## 1. 데스크톱 레이아웃 (1200px+)
```
+-------------------------------------------------------------+
|                      HEADER (sticky, 72px)                   |
|  [로고]                        [장바구니 (3)] [로그인]        |
+-------------------------------------------------------------+
|                                                               |
|  +--------+ +--------+ +--------+ +--------+                |
|  | 1위    | | 2위    | | 3위    | | 4위    |  4열 그리드     |
|  | 상품   | | 상품   | | 상품   | | 상품   |                |
|  +--------+ +--------+ +--------+ +--------+                |
|  +--------+ +--------+ +--------+ +--------+                |
|  | 5위    | | 6위    | | 7위    | | 8위    |                |
|  +--------+ +--------+ +--------+ +--------+                |
|                   ... (100개까지)                             |
+-------------------------------------------------------------+
|                      FOOTER (240px)                          |
|           사업자 정보 | 고객센터 | 면책 조항                  |
+-------------------------------------------------------------+
```

## 2. 모바일 레이아웃 (<768px)
```
+------------------------+
|    HEADER (56px)       |
| [로고]  [cart][login]  |
+------------------------+
| +-----+ +-----+       |
| |1위  | |2위  | 2열    |
| +-----+ +-----+       |
| +-----+ +-----+       |
| |3위  | |4위  |       |
| +-----+ +-----+       |
+------------------------+
|      FOOTER            |
+------------------------+
```

## 3. 헤더 상세
- 높이: 72px (데스크톱) / 56px (모바일)
- 배경: #FFFFFF, 하단 테두리: 1px solid #E0E0E0
- position: sticky, z-index: 100
- 로고: 20px 700, 색상 #2C7A3F
- 장바구니 아이콘: 24px, 뱃지: #E53935

## 4. 그리드 설정
- 데스크톱: repeat(4, 1fr), gap 20px, padding 24px
- 태블릿: repeat(3, 1fr), gap 16px
- 모바일: repeat(2, 1fr), gap 12px, padding 16px

## 5. 오버레이 z-index
- 100: 헤더, 200: 드롭다운, 300: 토스트, 900: 모달 오버레이, 910: 모달, 920: 슬라이드 패널
