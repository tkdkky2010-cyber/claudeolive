# 필수 면책 조항 및 표기 사항

**적용 대상**: 올리브영 랭킹 기반 커머스 사이트
**법적 근거**: 전자상거래법, 부정경쟁방지법, 소비자보호법

---

## 1. 메인 페이지 상단 경고 배너 (필수)

### 위치
- 모든 페이지 최상단 (헤더 직후)
- 모바일에서도 가독성 확보

### 권장 문구
```
본 사이트는 CJ올리브영(주)의 공식 사이트가 아니며, 제휴 관계가 없습니다.
올리브영에서 구매한 정품을 해외로 수출하는 독립 셀러가 운영하는 사이트입니다.
```

### 디자인
- 배경색: #FFF9E6 (연한 노란색)
- 텍스트 색: #333333
- 폰트 크기: 14px (모바일 12px)
- 패딩: 상하 12px, 좌우 20px

## 2. 푸터 필수 표기 사항

### 2.1 사업자 정보 (전자상거래법 제13조)
```
상호(법인명): (주)OOO
대표자: 홍길동
사업자등록번호: 000-00-00000
통신판매업 신고번호: 제0000-서울OO-00000호
주소: 서울특별시 OO구 OO로 00
이메일: contact@example.com
전화: 02-0000-0000
```

### 2.2 상품 관련 면책 조항
- 본 사이트의 모든 상품은 올리브영에서 구매한 정품
- 병행수입을 통해 유통되는 정품, 제조사 공식 수입 경로와 무관
- 병행수입 특성상 제조사 국내 공식 A/S 미적용 가능
- 랭킹 정보는 1일 1회 업데이트, 실시간 아님
- CJ올리브영(주)과 어떠한 공식 관계도 없음

### 2.3 저작권 고지
```
Copyright (c) 2026 (주)OOO. All rights reserved.
```
