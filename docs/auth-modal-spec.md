# 로그인 모달 상세 스펙

## 구조
```
+---------------------------+
|    [X 닫기]               |
|                           |
|        [로고 아이콘]       |
|     올영 TOP 100          |
|                           |
| "간편하게 로그인하고       |
|  장바구니를 이용하세요"    |
|                           |
| +------------------------+|
| | [G] 구글로 로그인       ||
| +------------------------+|
|                           |
| "로그인 시 이용약관 및     |
|  개인정보처리방침에        |
|  동의하는 것으로 간주"     |
+---------------------------+
        400px x auto
```

## 모달 외곽
- 크기: 400px (모바일: calc(100vw - 32px))
- 배경: #FFFFFF
- 테두리 반경: 12px
- box-shadow: var(--shadow-xl)
- z-index: 910

## 오버레이
- 배경: rgba(0, 0, 0, 0.5)
- z-index: 900
- 클릭 시 모달 닫힘
- 애니메이션: opacity 0→1, 0.3s

## 모달 등장 애니메이션
- scale(0.95)→scale(1) + opacity 0→1
- 0.3s cubic-bezier(0.4, 0, 0.2, 1)

## 닫기 버튼
- 위치: absolute, top: 16px, right: 16px
- 크기: 36x36px, 아이콘 20px
- 색상: #666666, 호버: #1A1A1A

## 로고 영역
- 아이콘: 48px
- 텍스트: 24px/700, 색상 #2C7A3F
- 간격: 아이콘-텍스트 12px

## 안내 문구
- 폰트: 14px/400, 색상 #666666
- 정렬: 중앙
- 마진: 24px 상하

## 구글 로그인 버튼
- 너비: 100%, 높이: 52px
- 배경: #FFFFFF, 테두리: 1px solid #E0E0E0
- 테두리 반경: 4px
- 구글 아이콘: 20px (좌측)
- 텍스트: "구글로 로그인", 16px/500, 색상 #1A1A1A
- 호버: 배경 #F8F8F8, 테두리 #CCCCCC
- 활성: 배경 #F0F0F0

## 하단 동의 텍스트
- 폰트: 12px/400, 색상 #999999
- 정렬: 중앙
- 마진 상단: 24px
- "이용약관" 및 "개인정보처리방침" 링크: 색상 #5C6AC4, 밑줄

## OAuth 플로우
1. 구글 로그인 버튼 클릭
2. 구글 OAuth 팝업 열림
3. 사용자 인증 완료
4. 모달 닫힘 (0.3s)
5. 헤더 변경: 로그인 버튼 → 사용자 프로필 (원형 아바타 36px + 이름)

## 접근성
- ESC 키: 모달 닫기
- Tab: 포커스 트랩 (모달 내부에서만 순환)
- aria-modal: true
- aria-labelledby: 모달 제목

## 반응형
- 모바일: 너비 calc(100vw - 32px), 패딩 24px
- 데스크톱: 400px 고정, 패딩 32px
